# Meteo
Test: Echanges serveurs
Objectifs:
- Créer un échange avec un serveur pour obtenir la météo d'une ville
- Renouveler cet échange toutes les 30 secondes avecune ville différente
- Afficher la météo graphiquement de manière claire

Comment l'utiliser:
  1- télécharger le dossier app
  2- ouvrir l'invite de commande et se rendre dans ce dossier
  3- taper les commandes suivantes:
     npm install node.js
     npm install unirest
     npm install express
     npm install http
     npm install socket.io
     npm install fs
     node server.js
  4- ouvrir un navigateur internet de votre choix et écrire dans la barre d'adresse web:
     localhost:3000
